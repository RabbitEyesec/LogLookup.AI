Welcome

LogLookup connects to your existing SIEM and groups related alerts into attack chains. It can review each chain, cite the evidence behind its verdict, and write the result back to the SIEM.

  • Correlation before model analysis. Alert relationships come from the correlation engine, not the model.
  • Keep data local. When you use a local model, evidence stays on this machine.
  • Set up in the browser. Credentials are stored in the encrypted local store.

Setup takes about two minutes.

Connect your SIEM

Enter your Elasticsearch URL and API key. After the connection test, choose the index that contains your alerts.

Lab only: certificate verification is disabled. Traffic and credentials could be intercepted. Never use this option in production.

Configure AI

Choose a local or cloud model. You can change the provider later in Settings without restarting the service.
